Features

  • Multiple theme support:
    • Light theme (Custom OKLCH colors)
    • Dark theme
    • Cupcake theme
    • Corporate theme
  • Font family support:
    • Sarabun (Thai + Latin)
    • Prompt (Thai + Latin)
  • Theme persistence with localStorage
  • Smooth transitions between themes
  • Responsive design
  • Thai language support
  • Light/Dark mode toggle
  • Custom theme colors in OKLCH
